Hyper Text Markup Language (HTML)

History and Evolution of HTML

Hypertext Markup Language (HTML) is the standard markup language used for creating web applications and pages. Tim Berners-Lee initially developed HTML to solve his problem of having to switch between different computers in order to access different pieces of information at his workplace, the CERN labs. The concept of this system was the beginning of the world wide web’s fundamental language. The evolution of the HTML language is discussed below.

HTML 1 (1993)

The first version of HTML was mainly developed for users to share information on the web. Webpages developed at the time were static and heavily text-based as HTML did not support any forms of styling (fonts, backgrounds, images). The most notable tags in this version are:

  • Anchor <a>
  • Title <title>
  • Lists <ul>, <ol>,<dl> and <li>
  • Headings <h1>...<h6>

HTML 2 (1995)

This was the first official standard of HTML and was published in 1995. HTML 2 supported more browsers and contained all of the features from HTML1, along with some additional tags. Of note, the following tags were introduced in HTML 2:

  • DocType <!doctype>
  • Forms <form>
  • Tables <table>
  • Body <body>
  • Text Semantics <strong>
  • Images <img>

HTML 3.2 (1997)

At this stage, HTML gained strong following with growing demand for newer tags and capabilities. HTML 3.0 was a relatively massive improvement compared to previous versions, implementing many features. Most browsers were unable to keep up with the size of the overhaul and the version was discarded. Through the intervention of the World Wide Web Consortium (W3C), a lower-specification but standardised language was developed and subsequently rolled out, as HTML 3.2. In this version, the most notable tags were:

  • Styles <style>
  • Fonts <font>

HTML 4.0 (1999)

This version of HTML provided support for the use of external style sheets (CSS), offering more dynamic presentation mechanisms. As a result, the W3 Consortium eventually phased out many of HTML’s presentation elements. Most notably, HTML 4 introduced:

  • Buttons <button>
  • Field Sets <fieldset>
  • Object <object>

HTML 5 (2014)

This is the latest version of HTML and in this update, the language offers a variety of new features such as:

  • Navigation <nav>
  • Sections <section>
  • Meter <meter>
  • Headers <header> and Footers <footer>

These tags help structure content more efficiently and allows for better search engine results. This version also provided support for multimedia elements using the Audio <audio> and Video <video> tags. HTML 5 also allowed the development of more dynamic and interactive websites without the need of extra software/plugins. By limiting the need for external plugins, content could be delivered much faster compared to previous HTML versions.

HTML Elements

Lists

HTML lists can be used to present information in 3 different ways, they are:

  • Unordered list - A list of items presented in no particular order (marked as bullet points).
  • Ordered list - A list of items, in a numbered order.
  • Description list - A list of terms and their specific descriptions.

Unordered Lists

Unordered lists can be created using two tags, <ul> (defines an unordered list) and <li> (defines list items). The <ul> tag is declared first, before the list elements are entered between the <li> and </li> tags. The list is then closed with </ul> tag. Images

Images in HTML are defined using the <img> tag. This tag has two main attributes, "src" (source location of the image, online or on a local system) and "alt" (a short text describing the image,
that is displayed when the image fails to load). In addition to these attributes, the image can be resized and aligned to the author's preference using the "width and "height" attributes.
In order to alter the positioning of the image, CSS must be used as the "align attribute for the <img> tag is no longer supported in HTML 5. The image below demonstrates the use of the

Anchor

Using the Anchor <a> tag, hyperlinks can be defined within a HTML page. This tag can be used for either hyperlinks or images. The attributes for this tag are "href" (defines the link address of the webpage/image) and "target" (defines where to open the linked webpage/image). Video

The HTML Video <video> tag allows videos to be played on a webpage. The tag supports 3 video formats, MP4, WebM and Ogg. The tag has a number of optional attributes such as
"src" (video file source), "autoplay" (video starts playing when ready), "loop" (video restarts whenever playback is over),and many more. An example of a MP4 video can be found below.

Object

The <object> tag allows for an emebedded object within the HTML page. The tag can be used to embed media (audio and video files), PDF and Flash files. The main attributes of this HTML element is data, which specifies the URL of the resource to be embedded and type, specifying the type of file to be used. Meter

The HTML <meter> element can be used to represent either a scalar or fractional value within a given range. The main attributes for this tag are value - the current numeric value, min - lower bound of the rang and max - upper bound of the range. Optional attributes include low - the range that is considered to be a low value, high - the range that is considered to be a high value and optimum - specifies what the ideal value for the scale is.
